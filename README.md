## Intro
Opiniated front-end starter

## Technologies
* React
* Flux + EventEmitter
* page.js
* Webpack 2
* Babel | ES6
* Standard JS linting
* Stylus
* gsap
* MobileDetect

## Installation
`yarn install`

## Launch the project
`yarn run dev`

## Build for production
`yarn run build`
